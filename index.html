Button

El input de tipo button crea un botón simple que no realiza ninguna acción por sí mismo.

<input type="button" value="Click Me">

Checkbox

El input de tipo checkbox permite seleccionar o deseleccionar una opción.

<input type="checkbox"> Opción
Opción

Color

El input de tipo color permite al usuario seleccionar un color utilizando un selector de colores.

<input type="color">

Date

El input de tipo date permite seleccionar una fecha en un calendario.

<input type="date">

Datetime-local

El input de tipo datetime-local permite seleccionar una fecha y una hora sin zona horaria.

<input type="datetime-local">

Email

El input de tipo email espera que el usuario introduzca una dirección de correo electrónico válida.

<input type="email">

File

El input de tipo file permite al usuario seleccionar un archivo de su dispositivo.

<input type="file">

Hidden

El input de tipo hidden no es visible para el usuario, pero envía información al servidor.

<input type="hidden" value="dato oculto">

Image

El input de tipo image crea un botón gráfico con una imagen.

<input type="image" src="https://via.placeholder.com/150" alt="Submit Image">

Month

El input de tipo month permite seleccionar un mes y año.

<input type="month">

Number

El input de tipo number permite seleccionar un número dentro de un rango.

<input type="number">

Password

El input de tipo password oculta el texto ingresado.

<input type="password">

Radio

El input de tipo radio permite seleccionar una opción de un grupo de opciones.

<input type="radio" name="grupo"> Opción 1
Opción 1 Opción 2

Range

El input de tipo range permite seleccionar un valor dentro de un rango con un control deslizante.

<input type="range">

Reset

El input de tipo reset reinicia los campos de un formulario a sus valores predeterminados.

<input type="reset" value="Reiniciar formulario">

Search

El input de tipo search crea un campo de búsqueda especializado.

<input type="search">

Submit

El input de tipo submit envía los datos del formulario al servidor.

<input type="submit" value="Enviar">

Tel

El input de tipo tel acepta números de teléfono.

<input type="tel">

Text

El input de tipo text es el más común y permite ingresar texto libre.

<input type="text">

Time

El input de tipo time permite seleccionar una hora específica.

<input type="time">

URL

El input de tipo url espera una URL válida.

<input type="url">

Week

El input de tipo week permite seleccionar una semana específica del año.

<input type="week">
